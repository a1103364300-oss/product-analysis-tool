# 更新日志 (Changelog)

所有值得注意的变更都将记录在此文件。

本项目遵循[语义化版本控制](https://semver.org/lang/zh-CN/)和[Keep a Changelog](https://keepachangelog.com/)规范。

## [2.0.0] - 2024-01-31

### 新增功能

#### 四大核心分析方法
- **四象限分析法**：基于毛利和销量中位数将商品分为四个象限，提供直观的商品分类视图
- **综合评分法**：支持自定义权重，计算综合得分并评定S/A/B/C/D五个等级
- **ABC分类法**：基于帕累托法则进行毛利和销量ABC分类，提供9宫格交叉分析
- **聚类分析**：使用K-means算法自动发现商品自然分组

#### 多维度雷达图分析
- 6个关键指标可视化：盈利能力、销售热度、综合评分、利润占比、销量占比、价值系数
- 三种查看模式：TOP10商品对比、各等级平均表现、自定义单商品分析
- 详细的评估指标说明，帮助理解每个维度含义

#### 可视化图表
- 四象限分布散点图（坐标原点在中位数）
- 评分分布直方图
- 毛利-销量关系散点图（按等级着色）
- 聚类分布散点图

### 功能改进

- 响应式设计，支持桌面端和移动端
- 优化的UI/UX，更现代的视觉风格
- 实时权重调整，综合评分法支持动态调整毛利和销量权重
- 数据处理在本地完成，保护数据隐私
- 支持CSV格式报告导出

### 技术栈升级

- 基于Chart.js 4.x的全新图表引擎
- SheetJS 0.19.x提供更强的Excel处理能力
- 完全的前端架构，无需服务器支持
- 支持Docker容器化部署

### 文档更新

- 完整的README文档，包含详细使用说明
- 贡献指南文档
- 部署指南文档

## [1.0.0] - 2024-01-15

### 初始发布

- **基础四象限分析功能**
  - Excel文件上传和解析
  - 基于中位数的四象限分类
  - 可视化散点图展示
  - 数据表格展示
  - CSV报告导出

### 核心功能

- 支持标准Excel格式（.xlsx）
- 自动识别数据列：商品名称、规格、SKU、总销量、商品毛利额
- 分类统计：各象限商品数量和占比
- 中位数自适应：自动计算阈值，无需手动设置

## 版本格式说明

本项目版本格式遵循[语义化版本2.0.0](https://semver.org/)：

```
主版本号.次版本号.修订号
```

- **主版本号**：不兼容的API变更
- **次版本号**：向后兼容的功能新增
- **修订号**：向后兼容的问题修复

## 版本标签说明

- **[新增]**: 新功能或特性
- **[改进]**: 功能优化或性能提升
- **[修复]**: Bug修复
- **[文档]**: 文档更新
- **[废弃]**: 即将移除的功能
- **[安全]**: 安全相关的修复或改进
